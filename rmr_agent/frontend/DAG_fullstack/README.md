In the project directory, you should install following packages:

# `pip install pyyaml fastapi uvicorn pyyaml pydantic`


Make sure you have Node.js and NVM in your system. I use node v12.22.1, nvm 0.39.3, npm 6.14.12
Remember to switch the path to your local path

Then run:
# `cd frontend`
# `npm install`

and then you could run the call_api.ipynb file directly. UI would pop out.

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

The page will reload when you make changes. And the verified_yaml.yaml will be updated at the same time.
You may also see any lint errors in the console.
